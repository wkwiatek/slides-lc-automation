Master your tools

npm as package manager

  • consistent dependencies
  • lock
  • works flawlessly with browserify
  • flat deps tree (v3)

npm

  • can also be a build tool
  • direct bash commands

Example

$ npm install jshint --save-dev

"scripts": {
  "info": "echo 'npm as a build tool'",
  "jshint": "jshint src/*.js"
}

grunt

  • configuration over code
  • ease of use
  • sequential-first
  • I/O intensive

Example

$ npm install grunt-contrib-jshint --save-dev

module.exports = function(grunt) {
  grunt.loadNpmTasks('grunt-contrib-jshint');

  grunt.initConfig({
    jshint: {
      all: 'src/*.js'
    }
  });
}

gulp

  • stream based
  • node.js knowledge required
  • concurrent-first

Example

$ npm install gulp-jshint --save-dev

var gulp = require('gulp');
var jshint = require('gulp-jshint');

gulp.task('jshint', function() {
    gulp.src('src/*.js')
      	.pipe(jshint())
        .pipe(jshint.reporter('default'));
});

webpack

  • configuration over code again
  • loaders for everything
  • hot module replacement

Example

$ npm install jshint-loader --save-dev

module.exports = {
  module: {
    preLoaders: [
      {
        test: /\.js$/,
        exclude: /node_modules/,
        loader: 'jshint-loader'
      }
    ]
  }
}
